## Welcome

A [Baselined.ai](https://www.baselined.ai/) account is required to create your API key for authorization. If you do not already have one, please [create an account](https://www.baselined.ai/register). Once registered, you will be able to view your API key on your [dashboard](https://www.baselined.ai/dashboard).

All API endpoints are authenticated using either an API Key or a Bearer token. You can find your credentials on the [Baselined.ai Developer Dashboard](https://www.baselined.ai/dashboard).
